Quem somos 

O projeto de extensão Interfaces Livres, iniciado em 2012, tem como objetivo geral criar mídias e projetos de tecnologias de informação e comunicação para fomentar a inclusão digital e a distribuição livre de técnicas, informação e conhecimento.

Ações da Extensão

Em desenvolvimento

Ciclo de Palestras Login

As palestras e minicursos Log{In} são uma iniciativa do projeto Interfaces Livres para apresentar os trabalhos de profissionais do mercado e pesquisadores que atuam dentro dos temas de interesse do projeto e das mídias digitais. Trata-se de uma ação contínua do projeto vinculada ao canal do Youtube do Interfaces Livres.

Socorro Pontos de Vida

A Socorro é um chatbot em desenvolvimento pelo interfaces livres. Tem o objetivo de incentivar e facilitar a doação de sangue na cidade de João Pessoa. Idealizada para representar uma das trabalhadoras do Hemocentro, Socorro já está disponível através do telegram e traz formas de agendar lembretes e convidar até grupos de amigos para realizar as doações.

Minetest Server

Instalação do software livre Minetest Server em um servidor dedicado através das instruções obtidas na documentação oficial. O servidor dedicado vem com a configuração padrão para iniciar junto com o boot do sistema, mas para isso dar certo é preciso seguir as configurações padrão desse formato, que incluem o posicionamento adequado para os arquivos de configuração e instalação.

Desenvolvidos

Moodle Classes

O sistema de ensino remoto utilizado pela Universidade Federal da Paraíba foi desenvolvido com base no ambiente virtual de aprendizagem de código aberto Moodle. A interface gráfica de exibição desse sistema foi customizada pelo projeto Interfaces Livres.

Caminho Livre UFPB

O projeto Caminho Livre surgiu a partir da percepção da não existência de informações relacionadas à acessibilidade dos ambientes da UFPB assim como em demais sistemas de localização e transporte. Visto isso, foi iniciado o projeto com objetivo de disponibilizar essas informações de forma ágil para o público.

PWA do Pontos de Vida

Progressive web app criado com objetivo de incentivar a doação voluntária e regular de sangue. A aplicação possui vários recursos para promover a doação e motivar o doador. Registro de doações, através do QR Code, lembrete para a próxima doação, conquista de níveis, figurinhas informativas, formação de grupos de doação, entre outros.

Teste de Estilo de Aprendizagem Kolb

O questionário idealizado por David A. Kolb tem como objetivo identificar e descrever diferentes estilos de aprendizagem.  O formulário desenvolvido apresenta os resultados do teste de Kolb e ajuda professores e alunos a escolher as melhores estratégias de ensino/aprendizagem.

Calculadora de Notas

A calculadora de notas desenvolvida pelo Interfaces Livres agiliza o cálculo referente à média e exame final. O sistema informa o resultado da média das notas e informa se o estudante está aprovado ou não. Caso a nota seja insuficiente, a calculadora informa também quanto será necessário tirar no exame final para conquistar a aprovação.

Podcast Ambiente 68

O podcast desenvolvido pelo Interfaces Livres traz vários episódios no estilo “discussão e mesa redonda” acerca de temas relacionados com a produção de diferentes tipos de conteúdos, desde UI/UX até desenvolvimento de quadrinhos autorais e sobre a temas como a cultura LGBTQIA+ dentro das mídias digitais.

Template da Marca de Fantasia

O tema para a editora Marca de Fantasia foi desenvolvido para oferecer mais dinamicidade e escalabilidade na atualização e gestão do conteúdo da editora criada em 1995 pelo professor Henrique Magalhães.

Missão Diploma (Jogo)

O jogo Missão Diploma constituiu uma experiência de desenvolvimento de um jogo plataformer que recria a experiência de conseguir um título de graduação na UFPB. Incluindo os desafios de recolher os documentos de matrícula e lidar com a burocracia para obter os resultados.
